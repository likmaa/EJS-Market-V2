# Logos de Paiement

## Instructions

Téléchargez les logos officiels des moyens de paiement et placez-les dans ce dossier.

### Logos nécessaires :

1. **visa.svg** ou **visa.png** - Logo Visa
   - Source officielle : https://brand.center/mastercard/fr/media-resources.html
   - Ou : https://www.visa.fr/about-visa/newsroom/media-kit.html

2. **mastercard.svg** ou **mastercard.png** - Logo Mastercard
   - Source officielle : https://brand.center/mastercard/fr/media-resources.html

3. **paypal.svg** ou **paypal.png** - Logo PayPal
   - Source officielle : https://www.paypal.com/us/webapps/mpp/logo-center

4. **amex.svg** ou **amex.png** - Logo American Express
   - Source officielle : https://www.americanexpress.com/fr-fr/content/brand-centre/

5. **stripe.svg** ou **stripe.png** - Logo Stripe
   - Source officielle : https://stripe.com/about/branding

### Alternative : Utiliser un CDN

Si vous ne souhaitez pas télécharger les logos, le code utilise automatiquement des URLs de CDN publics.
